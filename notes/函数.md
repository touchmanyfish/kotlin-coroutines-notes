# 函数

### function literals
lambda表达式和匿名函数都是[function literals](https://kotlinlang.org/docs/lambdas.html#lambda-expressions-and-anonymous-functions)

```kotlin
//lambda表达式形式的function literals
{ a, b -> a + b }

//匿名函数形式的function literals
fun(s: String): Int { return s.toIntOrNull() ?: 0 }
```

### Non-literal values of function types
从文档上看“Non-literal values of function types”意思就是“一个类型为函数的变量”
```kotlin
// twoParameters就叫做Non-literal values of function types
val twoParameters: (String, Int) -> String = repeatFun
```

### 带receiver和不带receiver的函数类型可相互交换使用的情况
f1:receiver类型 参数1类型 参数2类型  
f2:参数1类型    参数2类型 参数3类型  
如果这2个列表一致，则f1和f2可以[相互交换使用](https://kotlinlang.org/docs/lambdas.html#instantiating-a-function-type)

```kotlin
var f1: String.(Int, Int) -> Int = { x, y -> 100 }
var f2: (String, Int, Int) -> Int = f1
f1 = f2
```

### 将非匿名函数赋值给变量
| 类型 | 示例代码 |
|------|--------------------------------------------------|
| 普通函数引用 | `val funSum: (Int, Int) -> Int = ::sum` |
| 类的成员函数引用 | `val funMultiply: (Int, Int) -> Int = calculator::multiply` |
| 扩展函数引用 | `val funGreet: (String) -> String = String::greet` |

[Instantiating a function type﻿
](https://kotlinlang.org/docs/lambdas.html#lambda-expressions-and-anonymous-functions)

```kotlin
// 普通函数引用
val funSum = ::sum
funSum(100,200)

val calculator = Calculator()
// 类的成员函数引用
val funMultiply = calculator::multiply
funMultiply(100)
    
val funGreet = String::greet
// 扩展函数引用
funGreet("")
```
